En este directorio se encuentran los archivos de código del proyecto.
Esta dividido en varios directorios y subdirectorios, cada uno representado por un package.

>DATA: Este directorio contiene los archivos de texto que funcionan como base de datos.

>Domini: En este directorio se encuentran los archivos que representan las clases de la capa de dominio, con cualquier código auxiliar que
hayamos utilizado.

>Persistencia: Este directorio contiene las clases necesarias para la capa de persistencia. Esto incluye un controlador y clases para gestionar

>Presentacio: Este directorio contiene las clases necesarias para la capa de presentacion. Esto incluye un controlador y vistas para gestionar

>JUnit: Este directorio contiene los Unit Tests de las clases de la capa de dominio.

>Main: Este archivo contiene el main del proyecto, que permite la ejecución del programa.

>lib : Contiene las librerías necesarias para la correcta ejecución del programa.

>Utils: Directorio de clases de utilidad que se usan en distintos puntos del programa.
